This demo demonstrates ID generation and ID-based sampling, as specified in the EPC-ID specification.

Bucketing settings:

The identifier will be placed into one of four buckets, either with equal weights or unequal. If unequal (preset) weights are used, we will expect the proportion of identifiers in each bucket to be approximately 40%, 10%, 30%, 20%. Use equal weights?
